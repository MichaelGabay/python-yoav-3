# תרגילי קלט ופלט בפייתון

## תרגיל 1: הצגת שם

כתוב תוכנית שמבקשת מהמשתמש להכניס את שמו ומדפיסה הודעה: "שלום [שם], ברוך הבא!"

**דוגמה:**

```
Input: Yoav
Output: Hello Yoav, welcome!
```

---

## תרגיל 2: חישוב גיל

כתוב תוכנית שמבקשת מהמשתמש להכניס את שנת הלידה שלו ומדפיסה את הגיל שלו (הנח שהשנה הנוכחית היא 2026).

**דוגמה:**

```
Input: 2000
Output: Your age is: 26
```

---

## תרגיל 3: חישוב ממוצע

כתוב תוכנית שמבקשת מהמשתמש להכניס שני מספרים ומדפיסה את הממוצע שלהם.

**דוגמה:**

```
Input: 10
Input: 20
Output: The average is: 15.0
```

---

## תרגיל 4: הודעה מותאמת אישית

כתוב תוכנית שמבקשת מהמשתמש להכניס את שמו, העיר שבה הוא גר, וגילו, ומדפיסה הודעה מותאמת אישית.

**דוגמה:**

```
Input: Danny
Input: Tel Aviv
Input: 25
Output: Hello Danny! You live in Tel Aviv and you are 25 years old.
```

---

## תרגיל 5: חישוב שטח מלבן

כתוב תוכנית שמבקשת מהמשתמש להכניס את אורך ורוחב של מלבן (במטרים) ומדפיסה את השטח שלו.

**דוגמה:**

```
Input: 5
Input: 3
Output: The area of the rectangle is: 15 square meters
```

---

## פתרונות

### פתרון תרגיל 1:

```python
name = input("הכנס את שמך: ")
print(f"שלום {name}, ברוך הבא!")
```

### פתרון תרגיל 2:

```python
birth_year = int(input("הכנס את שנת הלידה שלך: "))
age = 2026 - birth_year
print(f"הגיל שלך הוא: {age}")
```

### פתרון תרגיל 3:

```python
num1 = float(input("הכנס מספר ראשון: "))
num2 = float(input("הכנס מספר שני: "))
average = (num1 + num2) / 2
print(f"הממוצע הוא: {average}")
```

### פתרון תרגיל 4:

```python
name = input("הכנס את שמך: ")
city = input("הכנס את העיר שבה אתה גר: ")
age = input("הכנס את גילך: ")
print(f"שלום {name}! אתה גר ב{city} ואתה בן {age}.")
```

### פתרון תרגיל 5:

```python
length = float(input("הכנס את אורך המלבן במטרים: "))
width = float(input("הכנס את רוחב המלבן במטרים: "))
area = length * width
print(f"שטח המלבן הוא: {area} מטרים רבועים")
```
