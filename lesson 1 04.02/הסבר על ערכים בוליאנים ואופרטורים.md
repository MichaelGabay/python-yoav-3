## ערכים בוליאניים (Boolean)

### מה זה ערך בוליאני?

ערך בוליאני הוא ערך שיכול להיות רק אחד משניים: **True** (אמת) או **False** (שקר).

בפייתון, ערכים בוליאניים הם סוג נתונים מיוחד שמשמש לבדיקת תנאים והחלטות.

### דוגמאות בסיסיות:

```python
# code
# ערכים בוליאניים ישירים
is_student = True
is_working = False

print(is_student)  # Output: True
print(is_working)  # Output: False
```

### איך נוצרים ערכים בוליאניים?

ערכים בוליאניים נוצרים בעיקר באמצעות:

- אופרטורים השוואתיים (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- אופרטורים לוגיים (`and`, `or`, `not`)
- פונקציות שמחזירות True/False

```python
# code
# השוואות מחזירות ערך בוליאני
age = 20
is_adult = age >= 18  # True
is_teenager = age < 18  # False

print(f"האם מבוגר? {is_adult}")  # Output: האם מבוגר? True
print(f"האם נער? {is_teenager}")  # Output: האם נער? False
```

### המרה לערך בוליאני:

```python
# code
# המרה מפורשת
number = 5
bool_value = bool(number)  # True (כל מספר שאינו 0 הוא True)

zero = 0
bool_zero = bool(zero)  # False

text = "Hello"
bool_text = bool(text)  # True (מחרוזת לא ריקה היא True)

empty_text = ""
bool_empty = bool(empty_text)  # False (מחרוזת ריקה היא False)
```

## אופרטורים אריתמטיים

אופרטורים אריתמטיים משמשים לביצוע חישובים מתמטיים.

### רשימת אופרטורים אריתמטיים:

| אופרטור | שם             | דוגמה     | תוצאה |
| ------- | -------------- | --------- | ----- |
| `+`     | חיבור          | `5 + 3`   | `8`   |
| `-`     | חיסור          | `10 - 4`  | `6`   |
| `*`     | כפל            | `6 * 7`   | `42`  |
| `/`     | חילוק          | `15 / 3`  | `5.0` |
| `//`    | חילוק שלם      | `15 // 4` | `3`   |
| `%`     | שארית (מודולו) | `15 % 4`  | `3`   |
| `**`    | חזקה           | `2 ** 3`  | `8`   |

### דוגמאות:

```python
# code
# חיבור וחיסור
a = 10
b = 3
sum_result = a + b  # 13
diff_result = a - b  # 7

print(f"סכום: {sum_result}")
print(f"הפרש: {diff_result}")
```

```python
# code
# כפל וחילוק
x = 8
y = 2
product = x * y  # 16
quotient = x / y  # 4.0

print(f"מכפלה: {product}")
print(f"מנה: {quotient}")
```

```python
# code
# חילוק שלם ושארית
num1 = 17
num2 = 5
integer_division = num1 // num2  # 3 (חלק שלם)
remainder = num1 % num2  # 2 (שארית)

print(f"חילוק שלם: {integer_division}")
print(f"שארית: {remainder}")
```

```python
# code
# חזקה
base = 2
exponent = 8
power = base ** exponent  # 256

print(f"{base} בחזקת {exponent} = {power}")
```

### סדר פעולות (PEMDAS):

פייתון מבצע פעולות לפי סדר עדיפות:

1. **P**arentheses (סוגריים) - `()`
2. **E**xponents (חזקות) - `**`
3. **M**ultiplication, **D**ivision (כפל, חילוק) - `*`, `/`, `//`, `%`
4. **A**ddition, **S**ubtraction (חיבור, חיסור) - `+`, `-`

```python
# code
# דוגמה לסדר פעולות
result1 = 2 + 3 * 4  # 14 (לא 20!)
result2 = (2 + 3) * 4  # 20
result3 = 2 ** 3 + 1  # 9 (לא 16!)
result4 = 10 / 2 * 3  # 15.0

print(f"2 + 3 * 4 = {result1}")
print(f"(2 + 3) * 4 = {result2}")
print(f"2 ** 3 + 1 = {result3}")
print(f"10 / 2 * 3 = {result4}")
```

## אופרטורים השוואתיים

אופרטורים השוואתיים משמשים להשוואה בין ערכים ומחזירים ערך בוליאני (True או False).

### רשימת אופרטורים השוואתיים:

| אופרטור | שם              | דוגמה    | תוצאה  |
| ------- | --------------- | -------- | ------ |
| `==`    | שווה ל-         | `5 == 5` | `True` |
| `!=`    | שונה מ-         | `5 != 3` | `True` |
| `<`     | קטן מ-          | `3 < 5`  | `True` |
| `>`     | גדול מ-         | `5 > 3`  | `True` |
| `<=`    | קטן או שווה ל-  | `5 <= 5` | `True` |
| `>=`    | גדול או שווה ל- | `5 >= 3` | `True` |

### דוגמאות:

```python
# code
# בדיקת שוויון
age1 = 20
age2 = 20
are_equal = age1 == age2  # True

print(f"האם הגילאים שווים? {are_equal}")
```

```python
# code
# בדיקת אי-שוויון
num1 = 10
num2 = 15
are_different = num1 != num2  # True

print(f"האם המספרים שונים? {are_different}")
```

```python
# code
# השוואות גדול/קטן
score = 85
passing_grade = 60

is_passing = score >= passing_grade  # True
is_excellent = score > 90  # False

print(f"האם עבר? {is_passing}")
print(f"האם מצוין? {is_excellent}")
```

```python
# code
# השוואות עם מספרים
x = 5
y = 10

less_than = x < y  # True
greater_than = x > y  # False
less_or_equal = x <= 5  # True
greater_or_equal = y >= 10  # True

print(f"{x} < {y}? {less_than}")
print(f"{x} > {y}? {greater_than}")
print(f"{x} <= 5? {less_or_equal}")
print(f"{y} >= 10? {greater_or_equal}")
```

```python
# code
# השוואות עם מחרוזות (לקסיקוגרפית)
name1 = "Alice"
name2 = "Bob"

is_before = name1 < name2  # True (A לפני B באלפבית)
is_after = name1 > name2  # False

print(f"'{name1}' לפני '{name2}'? {is_before}")
```

## אופרטורים לוגיים

אופרטורים לוגיים משמשים לשילוב מספר תנאים ומחזירים ערך בוליאני.

### רשימת אופרטורים לוגיים:

| אופרטור | שם  | תיאור                                          | דוגמה            | תוצאה   |
| ------- | --- | ---------------------------------------------- | ---------------- | ------- |
| `and`   | וגם | מחזיר True רק אם שני התנאים נכונים             | `True and False` | `False` |
| `or`    | או  | מחזיר True אם לפחות אחד התנאים נכון            | `True or False`  | `True`  |
| `not`   | לא  | הופך את הערך הבוליאני (True→False, False→True) | `not True`       | `False` |

### טבלת אמת (Truth Table):

#### אופרטור `and`:

| A     | B     | A and B |
| ----- | ----- | ------- |
| True  | True  | True    |
| True  | False | False   |
| False | True  | False   |
| False | False | False   |

#### אופרטור `or`:

| A     | B     | A or B |
| ----- | ----- | ------ |
| True  | True  | True   |
| True  | False | True   |
| False | True  | True   |
| False | False | False  |

#### אופרטור `not`:

| A     | not A |
| ----- | ----- |
| True  | False |
| False | True  |

### דוגמאות:

```python
# code
# אופרטור AND - וגם
age = 25
has_license = True

can_drive = age >= 18 and has_license  # True (שני התנאים נכונים)

print(f"האם יכול לנהוג? {can_drive}")
```

```python
# code
# אופרטור OR - או
grade1 = 85
grade2 = 55

passed_at_least_one = grade1 >= 60 or grade2 >= 60  # True (אחד מהם עבר)

print(f"האם עבר לפחות אחד? {passed_at_least_one}")
```

```python
# code
# אופרטור NOT - לא
is_raining = False
is_sunny = not is_raining  # True

print(f"האם יורד גשם? {is_raining}")
print(f"האם שמש? {is_sunny}")
```

```python
# code
# שילוב אופרטורים לוגיים
age = 20
has_ticket = True
is_weekend = False

can_enter = age >= 18 and has_ticket  # True
can_enter_free = age < 18 or is_weekend  # False

print(f"האם יכול להיכנס? {can_enter}")
print(f"האם כניסה חינם? {can_enter_free}")
```

```python
# code
# דוגמה מורכבת יותר
score = 75
attendance = 0.9  # 90%
is_final_exam = True

passed = score >= 60 and attendance >= 0.8  # True
exempt_from_final = score >= 90 or (score >= 80 and not is_final_exam)  # False

print(f"האם עבר? {passed}")
print(f"האם פטור מבחינה? {exempt_from_final}")
```

## שילוב אופרטורים

ניתן לשלב אופרטורים אריתמטיים, השוואתיים ולוגיים יחד בביטויים מורכבים.

### סדר עדיפות אופרטורים:

1. **סוגריים** `()` - תמיד ראשונים
2. **חזקות** `**`
3. **כפל, חילוק, שארית** `*`, `/`, `//`, `%`
4. **חיבור, חיסור** `+`, `-`
5. **השוואות** `<`, `>`, `<=`, `>=`, `==`, `!=`
6. **לוגיים** `not`, `and`, `or`

### דוגמאות לשילוב:

```python
# code
# שילוב אריתמטי והשוואתי
num1 = 10
num2 = 5
num3 = 3

average = (num1 + num2 + num3) / 3
is_above_average = average > 7  # True

print(f"הממוצע הוא: {average}")
print(f"האם מעל 7? {is_above_average}")
```

```python
# code
# שילוב אריתמטי, השוואתי ולוגי
age = 25
salary = 5000
years_experience = 3

is_qualified = age >= 21 and salary > 4000  # True
is_senior = years_experience >= 5 or salary >= 8000  # False

print(f"האם כשיר? {is_qualified}")
print(f"האם בכיר? {is_senior}")
```

```python
# code
# ביטוי מורכב עם סוגריים
x = 10
y = 5
z = 3

result1 = x + y * z  # 25 (לא 45!)
result2 = (x + y) * z  # 45
is_valid = (x + y) > 10 and z < 5  # True

print(f"x + y * z = {result1}")
print(f"(x + y) * z = {result2}")
print(f"האם תקף? {is_valid}")
```

```python
# code
# בדיקת טווח מספרים
number = 15
is_in_range = number >= 10 and number <= 20  # True
is_outside_range = number < 10 or number > 20  # False

print(f"האם בטווח 10-20? {is_in_range}")
print(f"האם מחוץ לטווח? {is_outside_range}")
```

```python
# code
# בדיקת מספר זוגי
num = 8
is_even = num % 2 == 0  # True (השארית בחלוקה ב-2 היא 0)
is_odd = num % 2 != 0  # False

print(f"האם זוגי? {is_even}")
print(f"האם אי-זוגי? {is_odd}")
```

```python
# code
# דוגמה מעשית - בדיקת כשירות לנהיגה
age = 22
has_license = True
has_insurance = True
is_sober = True

can_drive = age >= 18 and has_license and has_insurance and is_sober  # True

print(f"האם יכול לנהוג? {can_drive}")
```

```python
# code
# דוגמה מעשית - בדיקת ציון מעבר
test_score = 75
homework_score = 80
project_score = 90

average = (test_score + homework_score + project_score) / 3
passed = average >= 60  # True
excellent = average >= 90  # False
needs_improvement = average < 70  # False

print(f"הממוצע: {average}")
print(f"עבר? {passed}")
print(f"מצוין? {excellent}")
print(f"צריך שיפור? {needs_improvement}")
```

## טיפים חשובים

1. **סדר פעולות** - תמיד השתמש בסוגריים כדי להבהיר את הכוונה שלך, גם אם זה לא הכרחי.

2. **קריאות קוד** - ביטויים מורכבים קשים לקריאה. לפעמים עדיף לפרק אותם למשתנים ביניים:

   ```python
   # code
   # ❌ קשה לקריאה
   result = (x + y) * z > 10 and (a - b) < 5

   # ✅ יותר קריא
   sum_xy = x + y
   product = sum_xy * z
   diff_ab = a - b
   condition1 = product > 10
   condition2 = diff_ab < 5
   result = condition1 and condition2
   ```

3. **המרות אוטומטיות** - פייתון מבצעת המרות אוטומטיות בעת השוואות:

   ```python
   # code
   # פייתון יכולה להשוות מספרים ומחרוזות
   num = 5
   text = "5"
   # num == text  # False (סוגים שונים)
   # int(text) == num  # True (לאחר המרה)
   ```

4. **Short-circuit evaluation** - אופרטורים לוגיים עוצרים ברגע שהתוצאה ידועה:

   ```python
   # code
   # אם הצד השמאלי של AND הוא False, הצד הימני לא נבדק
   # אם הצד השמאלי של OR הוא True, הצד הימני לא נבדק
   ```
