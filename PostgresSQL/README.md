## מסד הנתונים Northwind

Northwind הוא מסד נתונים לדוגמה של חברת סחר (מכירות, מוצרים, לקוחות, הזמנות). כל הדוגמאות והתרגילים בקורס מבוססים עליו. ביחידה [00-Northwind-Setup.md](00-Northwind-Setup.md) מפורט איך לטעון את המסד ולהכיר את הטבלאות.

---

## איך לעבוד עם הקורס

1. טען את מסד Northwind לפי ההוראות ב-[00-Northwind-Setup.md](00-Northwind-Setup.md).
2. עבור לפי הסדר מיחידה 1 ל-8.
3. בכל יחידה: קרא את ההסבר, הרץ את דוגמאות הקוד, ופתור את התרגילים בסוף.

בכל בלוק קוד מופיעה הערה שמתחילה ב־`code|` – זו תווית שמזהה את הדוגמה.
